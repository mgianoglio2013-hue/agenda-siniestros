# ⚡ Agenda Siniestros — Gianoglio Peritaciones

Sistema central de gestión de siniestros con IA integrada.

## Módulos
- **Agenda** — Panel central, todos los siniestros, pipeline de etapas
- **Dictamen IA** — Análisis de culpabilidad con Ley 24.449 + geo-verificación
- **Google Drive** — Persistencia automática de todos los datos

## Stack
- React + Vite (frontend)
- Cloudflare Pages (hosting)
- Railway (agente IA backend — próximamente)
- Google Drive API (base de datos)
- Claude API (inteligencia)

## Portales integrados
MAPFRE · San Cristóbal · Integrity · Galeno · Provincia Seguros · OriongG2 · Valua

## Deploy
Conectado a Cloudflare Pages — push a `main` despliega automáticamente.
